# TeekaGramAi - Руководство по установке (⊙o⊙)

Это руководство поможет установить и настроить программу с нуля (≧∇≦)ﾉ

## Базовые зависимости
### Шаг 1: Установка Python

Для работы программы необходим Python версии 3.8 или новее

1.  Перейди на официальный сайт [python.org](https://www.python.org/downloads/) и скачай последнюю версию Python
2.  Запусти установщик
3.  **Очень важно:** На первом экране установки обязательно поставить галочку **"Add Python to PATH"**. Это позволит запускать Python из любой папки в командной строке, без этого никак! 

   ![Image](https://github.com/user-attachments/assets/ae4956be-3e92-4477-b889-cb607af0a4dc)
   
4.  Следуй инструкциям на экране до завершения установки []~(￣▽￣)~*
   (Где будет связанное с установкой pip, везде ставьте галочку что да, устаноить его. [Вот такой гайд есть на ютубе, чтобы вам было проще установить](outube.com/watch?v=82qj_kXQpuo))

5.  Чтобы проверить, что все установлено правильно, открой командную строку (или терминал) и выполни две команды:

    ```bash
    python --version
    pip --version
    ```

    Каждая команда должна вывести свою версию без ошибок.
#### **Как открыть командную строку (терминал)?**

Все дальнейшие команды нужно будет вводить в специальное окно. Вот как его открыть:

*   Нажмите клавишу `Win` (кнопка с флажком на клавиатуре), начните печатать `cmd` и выберите найденное приложение.

   ![Image](https://github.com/user-attachments/assets/158bdaf6-eaba-4b6c-82a4-3b63832e1db9)
   
### Шаг 2: Скачивание программы

1.  Вам нужно скачать все файлы программы. Самый простой способ - клонировать репозиторий с помощью Git. Если у вас не установлен Git, вы можете скачать его [здесь](https://git-scm.com/downloads)

    ```bash
    git clone https://github.com/biolog-end/TeekaGramAi
    ```

2.  Если у вас нет Git, вы можете просто скачать все файлы в виде ZIP-архива и распаковать его в удобную для вас папку ^^

   ![image](https://github.com/user-attachments/assets/88b3961e-1967-4a44-abad-d32bac6a4654)

### Шаг 3: Установка зависимостей

1.  Откройте папку в которую вы загрузили программу, и откройте в ней консоль (можно написать в строку поиска cmd, и нажать Enter, после чего откроется консоль)

   ![image](https://github.com/user-attachments/assets/53e5bdac-2013-4f34-8ab3-29fcd7c00fa7)

2.  Теперь установим все необходимые библиотеки с помощью одной команды. Эта команда установит все пакеты напрямую в вашу основную среду Python

    ```bash
    pip install -r requirements.txt
    ```

    Дождитесь завершения установки всех пакетов.

## Особенные для этого проекта шаги
## Шаг 4: Создание необходимых файлов и папок

Внутри папки с проектом вам нужно создать следующую структуру:

1.  Окройте папку `data`
2.  Внутри папки `data` перименуйте файл `accountsExample.json` в `accounts.json`. Он нужен для выбора аккаунта Telegram при запуске. Настройте в нём название аккаунтов которые вы хоятите иметь, и их количество:

    ```json
    {
      "Название акканута 1": "accounts/account1.session",
      "Название аккаунта 2": "accounts/account2.session"
    }
    ```
    *   `"Название акканута 1"` - это любое удобное вам имя для отображения в консоли
    *   `"account1"` - это имя файла сессии, который будет создан автоматически. Используйте только латинские буквы без пробелов, его будет видеть только программа по этому всё равно на него

## Шаг 5: Получение API ключей

Для работы программы нужны два набора ключей: от Telegram и от Google

### 5.1. Telegram API

1.  Перейдите на сайт [my.telegram.org](https://my.telegram.org) и войдите в свой аккаунт Telegram 
2.  После входа, нажми на **"API development tools"**
3.  Заполните короткую форму "Create new application" 
4. `App title`, `Short name` можно указать любые, например, `MyBrother` и `mybro42`. `URL` тоже не имеет значение, можно написать  
   ```txt
	http://localhost/
    ```
    `Platform` лучше выбрать `Desktop`, `Description` можно оставить пустым
	
    ![image](https://github.com/user-attachments/assets/cd380f09-eb9b-4ae6-9a0c-307e34a7b0b6)
	
5.  После создания приложения вы увидите ваши ключи: **`api_id`** и **`api_hash`**. Скопируй их, они нам понадобятся чуть позже
   (Данные на картинке - фейк)
   
   ![image](https://github.com/user-attachments/assets/7e0ab341-bc58-4909-8777-59d3b32cdff5)

### 5.2. Google Gemini API

1.  Перейдите на сайт [Google AI Studio](https://aistudio.google.com/)
2.  Войдите в свой Google-аккаунт
3.  Нажмите на кнопку **"Get API key"** в нижнем левом углу
4.  Нажмите **"Create API key"**
5.  Скопируйте сгенерированный ключ, обязательно сохранив где-нибудь! 

## Шаг 6: Конфигурация

Теперь нужно передать полученные ключи в программу.

1.  В корневой папке проекта (там же, где `main.py`) создай файл с именем `.env`
2.  Открой его и скопируйте туда следующий текст, вставив свои ключи вместо `...`:

    **Файл: `.env`**
    ```env
    # --- Ключи Telegram ---
    TELAGRAMM_API_ID="..."
    TELAGRAMM_API_HASH="..."

    # --- Ключ Google Gemini ---
    GOOGLE_API_KEY="..."

    # Номер инстанса, если вы запускаете несколько копий. Меняет порт веб-интерфейса
    INSTANCE_NUMBER=1
    ```

## Шаг 7: Первый запуск

Все готово к первому запуску! ヾ(≧▽≦*)o

1.  Запустите программу из папки проекта командой:

    ```bash
    python main.py
    ```

2.  При первом запуске в консоли произойдет следующее:
    *   Программа попросит вас выбрать аккаунт из тех, что вы указали в `data/accounts.json`. Введите цифру и нажмите Enter
    *   Далее программа попросит вас авторизоваться: введите номер телефона, код из Telegram и, если нужно, пароль двухфакторной аутентификации

3.  В консоли появится сообщение о том, что веб-сервер запущен. Обычно он доступен по адресу `http://127.0.0.1:5001` (порт зависит от `INSTANCE_NUMBER` в `.env` файле)
4.  Откройте этоу ссылку в браузере, чтобы начать работу

**Установка завершена! 🤑(∩^o^)⊃━☆**
